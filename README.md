# failure-lantern
A lightweight log triage utility that uses deterministic pattern matching and local LLM assistance to surface likely root causes from noisy application logs.

Failure Lantern helps engineers navigate noisy logs during production incidents. Instead of summarizing everything, it identifies likely failure chains, clusters related errors, and highlights the first observable fault condition.
